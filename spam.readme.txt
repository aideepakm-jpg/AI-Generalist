🔍 Project Workflow

1️⃣ Data Loading & Exploration
Load dataset
Check structure and distribution
Analyze spam vs ham ratio

2️⃣ Text Preprocessing
Convert text to lowercase
Remove punctuation and special characters
Remove stopwords
Tokenization
Stemming / Lemmatization

3️⃣ Feature Extraction
Convert text into numerical form using:
Bag of Words (CountVectorizer)
TF-IDF Vectorizer

4️⃣ Model Building
Machine learning models used:
Naive Bayes
Logistic Regression
Decision Tree (optional)
Random Forest (optional)

5️⃣ Model Evaluation
Accuracy Score
Confusion Matrix
Precision, Recall, F1-score
ROC-AUC Score

📊 Key Insights
Text preprocessing significantly improves model performance
TF-IDF usually performs better than basic count vectorization
Naive Bayes works well for text classification problems
Balanced dataset improves prediction quality
