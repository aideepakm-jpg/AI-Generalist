K-Means Clustering for Image Color Quantization
Developed by DEEPAK.M AI Generalist iPEC Solutions Private Limited Bangalore

🔍 Project Workflow

1️⃣ Load Image
Read image using OpenCV
Convert from BGR to RGB

2️⃣ Preprocessing
Reshape image into 2D array
Each row represents a pixel (R, G, B)

3️⃣ Apply K-Means
Choose number of clusters (K)
Fit K-Means model
Predict cluster labels

4️⃣ Recreate Image
Replace each pixel with its cluster centroid
Reshape back to original image dimensions

5️⃣ Visualization
Display:
Original Image
Compressed Image

📊 Key Insights
Higher K → Better image quality but less compression
Lower K → More compression but loss of detail
K-Means effectively identifies dominant colors
🖼️ Output
Compressed image with reduced colors
Visual comparison between original and processed image
